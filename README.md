## 🏥 ¿Qué es este sistema?

Este sistema clínico es una aplicación web desarrollada en **Laravel** que permite administrar de forma eficiente una clínica o consultorio médico.

Incluye módulos para gestionar pacientes, citas, historiales clínicos y usuarios del sistema.

---

## ✨ Características

### 👨‍⚕️ Gestión de Pacientes
- Registro de pacientes
- Historial médico
- Datos personales y de contacto
- Edición y eliminación

### 📅 Gestión de Citas
- Agendar citas médicas
- Calendario de consultas
- Control de asistencia

### 📄 Historial Clínico
- Diagnósticos
- Tratamientos
- Evolución del paciente

### 🔐 Usuarios y Roles
- Registro e inicio de sesión
- Roles (Administrador, Médico, Recepción)
- Control de acceso

### 💊 Gestión de Recetas (Opcional)
- Crear recetas médicas
- Historial de recetas
